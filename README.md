# SpendWise 💰

A modern expense tracker built with React and Tailwind CSS.

## 🚀 Features

- Add expenses
- Delete expenses
- Calculate total spending
- Persistent data using LocalStorage
- Responsive UI

## 🛠 Tech Stack

- React
- JavaScript
- Tailwind CSS
- Vite

## 🌐 Live Demo

https://spendwise-expense-tracker-blond.vercel.app

Through this project I learned:

- React component architecture
- Props and state management
- useEffect hook
- LocalStorage integration
- Deploying React applications
